ВИМОГИ:



1.юзер має можливість створити акаунт
  - нік повинен бути унікальним

  - пароль повинен бути безпечним

2.юзер повинен залогінитися для гри.

   - якщо неправильний пароль то повинно вискочити попередження

   - якщо правильний то все добре

3.вікно логіну повинно мати поле для введеня нікнейму, чекбокс ремембер мі, кнопку для реєстрації, кнопку для зміни юзера, видалення юзера.

4.юзер повинен зберігатися в базі даних.

  - його прогрес і всі ігри мовинні зберігатися

  - можливість відновлювати збережену гру

5.юзер повинен придумати захищений і безпечний пароль.

  - пароль повинен містити хоча б одну малу, одну велику латинські букви і цифри

  - мати довжину 5-12 символів

6.юзер повинен мати можливість додавати власні зображення.

  - додавати зображення в базу даних

  - можливість складання завантаених зображень

  - видаляти зображення які він додав

7.юзер повинен мати можливість зберігати і загружати збережену гру.

  - повинна бути кнопка save game

  - повинна бути кнопка load game

8.юзер повинен мати можливість пограти ставити на паузу.

  - під час гри можна поставити гру на паузу

  - повинне вискочити меню

  - можливість зберегти гру

9.юзер повинен мати можливість подивитися історію.

  - повинна бути можливість подивитися на свої зібрані пазли

  - можливість дивитися на час за який зібрали пазл

10.юзер повинен мати можливість видалити себе.

  - можливість видалення акаунту

  - після цього є можливість створити акаунт з видаленим ніком

11.можливість вибрати гру

  -вибрати розмір

  - вибрати картинку

  -вибрати режим гри
12. під час гри повинна бути кнопка паузи, меню, час який грає користувач, кількість ходів і картинка яку він складає.
13. технології
 
  - WPF

  - Postgres SQL

  - ASP .Net (C#)

14. Інтерфес повинен бути англійською
